# 图片资源目录

这个目录用于存放博客中使用的图片资源。

## 背景图片

要添加背景图片，请将图片文件命名为 `background.jpg` 并放置在此目录中。

推荐使用以下类型的图片：
- 淡色调的风景图片
- 抽象的纹理背景
- 低对比度的图片
- 确保文字在背景上清晰可读

### 图片要求：
- 格式：JPG、PNG 或 WebP
- 尺寸：建议至少 1920x1080 像素
- 文件大小：建议不超过 500KB
- 风格：简洁、淡雅，避免过于花哨

### 使用方法：
1. 将背景图片文件上传到此目录
2. 确保文件名为 `background.jpg`
3. 图片会自动应用到主页和文章页面的背景
